Hexo 基本介紹

結構：

  • node_module => 這個應該不用多做解釋了
  • public => 存放網站裡的內容，圖片，標籤等等
  • scaffolds => 網站的模板
  • themes => 存放網站的主題
  • _config.yml => 設定網站內容 （檔案中官方有詳細的註解）

基本指令

  • hexo new => 產生新的page，category，tag， draft等等 （詳細可以參考 https://hexo.io/docs/writing.html）
  • hexo clean => 將目前資料夾中所有網頁清除
  • hexo generate => 依設定檔產生網頁
  • hexo server => 在本機產生網頁，供使用者瀏覽
  • hexo deploy => 將目前的網頁發布上Ｇithub Page

  1. 執行 hexo deploy出現，github authentication error

    原因：

    Github在2021/8/12後，對跟git有關的存取不再支援password，改成使用token，有興趣可以看 https://github.blog/2020-12-15-token-authentication-requirements-for-git-operations/

    解決方法:

    前往Github Profile Developer Settings頁面，產生Token後作為Password貼上即可
